# .emacs.d

## Description
My Emacs configuration files.

## Requirement
- Emacs >= 24.5
- virtualenv

## Installation
### Basic

    $ git clone https://github.com/tsukachu/.emacs.d.vim ~/.emacs.d/
    $ cd ~/.emacs.d/el-get/
    $ git clone https://github.com/dimitri/el-get.git

Run Emacs.

### Lazy
Run Emacs.
